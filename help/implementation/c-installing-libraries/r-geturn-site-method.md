# getUrn-Site-Methode{#geturn-site-method}

Diese Methode gibt die URN für diese Site zurück.

## Java-Beispiel {#section_nyl_ycs_rz}

```

```

Beispielausgabe:

```
"urn:livefyre:network=`example.fyre.co`:site=1"
```
